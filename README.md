# GoNext

Минимальное приложение на Expo Router + TypeScript с React Native Paper.

## Установка

```powershell
npm install
```

## Запуск

```powershell
npm start
```

Затем выберите платформу:
- `a` для Android
- `i` для iOS
- `w` для Web

## Структура проекта

- `app/` - экраны приложения (Expo Router)
  - `_layout.tsx` - корневой layout с провайдером React Native Paper
  - `index.tsx` - экран Home
