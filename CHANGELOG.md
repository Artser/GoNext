# История изменений

## [1.0.0] - 2026-02-08

### Добавлено
- ✅ Базовая структура проекта на Expo Router + TypeScript
- ✅ Интеграция React Native Paper для UI компонентов
- ✅ Настройка базы данных SQLite с таблицами:
  - places (места)
  - trips (поездки)
  - trip_places (места в поездках)
  - place_photos (фотографии)
- ✅ Режим "Места":
  - Список мест с фильтрацией и поиском
  - Создание и редактирование мест
  - Детальная информация о месте
  - Добавление фотографий к местам
  - Открытие мест на карте и в навигаторе
- ✅ Режим "Поездки":
  - Список поездок с прогрессом
  - Создание и редактирование поездок
  - Добавление мест в поездку
  - Изменение порядка мест в маршруте
  - Отметка мест как посещенных
  - Добавление заметок и фотографий к посещениям
  - Переключение между планом и дневником
- ✅ Режим "Следующее место":
  - Автоматическое определение следующего места
  - Расчет расстояния до места
  - Быстрая отметка места как посещенного
- ✅ Главный экран с навигацией и статистикой
- ✅ Экран настроек с информацией о приложении
- ✅ Работа с фотографиями:
  - Выбор из галереи или создание фото камерой
  - Галерея фотографий с просмотром
  - Удаление фотографий
- ✅ Интеграция с картами:
  - Встроенные карты (OpenStreetMap через Leaflet)
  - Выбор координат на карте
  - Просмотр мест на карте
  - Просмотр всех мест поездки на карте
  - Открытие в Google Maps и системном навигаторе
- ✅ Валидация форм:
  - Валидация обязательных полей
  - Валидация координат
  - Валидация дат и диапазонов дат
- ✅ Обработка ошибок:
  - Централизованная обработка ошибок БД
  - Обработка ошибок файловой системы
  - Обработка ошибок геолокации
  - Понятные сообщения пользователю
- ✅ Оптимизация производительности:
  - Индексы в базе данных
  - Параллельные запросы где возможно

### Технические детали
- Expo SDK 52
- React Native 0.76
- TypeScript с строгой типизацией
- SQLite для локального хранения
- React Native Paper для UI
- OpenStreetMap для карт
- Полная поддержка офлайн режима
