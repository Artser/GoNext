### Рекомендации для обновления версий

Для гарантии совместимости рекомендуется использовать команду `npx expo install`:

```powershell
npx expo install expo-sqlite expo-image-picker expo-location expo-file-system expo-asset expo-font expo-status-bar
```

Эта команда автоматически установит версии пакетов, совместимые с вашей версией Expo SDK 52.
