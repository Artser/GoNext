# Руководство по тестированию GoNext

## Чеклист функционального тестирования

### Режим "Места"

#### Создание места
- [ ] Создать место с минимальными данными (только название)
- [ ] Создать место со всеми полями (название, описание, координаты, статусы)
- [ ] Попробовать создать место без названия (должна быть ошибка валидации)
- [ ] Создать место с неверными координатами (должна быть ошибка валидации)
- [ ] Использовать "Текущее местоположение" для получения координат
- [ ] Использовать "Выбрать на карте" для выбора координат

#### Редактирование места
- [ ] Открыть существующее место
- [ ] Изменить название
- [ ] Изменить описание
- [ ] Изменить координаты
- [ ] Изменить статусы (visitlater, liked)
- [ ] Сохранить изменения

#### Удаление места
- [ ] Удалить место из списка
- [ ] Удалить место из детального экрана
- [ ] Проверить, что место удалено из всех поездок (CASCADE)

#### Фотографии места
- [ ] Добавить фотографию из галереи
- [ ] Создать фотографию камерой
- [ ] Просмотреть фотографию в полноэкранном режиме
- [ ] Удалить фотографию

#### Фильтрация и поиск
- [ ] Отфильтровать места по статусу "Посетить"
- [ ] Отфильтровать места по статусу "Понравилось"
- [ ] Найти место по названию через поиск
- [ ] Обновить список (pull-to-refresh)

### Режим "Поездки"

#### Создание поездки
- [ ] Создать поездку с минимальными данными (только название)
- [ ] Создать поездку со всеми полями
- [ ] Попробовать создать поездку без названия (должна быть ошибка)
- [ ] Создать поездку с неверными датами (должна быть ошибка)
- [ ] Создать поездку с датой начала позже даты окончания (должна быть ошибка)
- [ ] Выбрать места из коллекции при создании
- [ ] Отметить поездку как текущую

#### Редактирование поездки
- [ ] Изменить название поездки
- [ ] Изменить даты поездки
- [ ] Добавить места в поездку
- [ ] Удалить места из поездки
- [ ] Изменить порядок мест в маршруте

#### Управление местами в поездке
- [ ] Отметить место как посещенное
- [ ] Добавить дату посещения
- [ ] Добавить заметки к посещенному месту
- [ ] Добавить фотографии к посещенному месту
- [ ] Изменить порядок мест (кнопки вверх/вниз)

#### Просмотр поездки
- [ ] Переключиться между режимами "План" и "Дневник"
- [ ] Просмотреть все места поездки на карте
- [ ] Проверить отображение прогресса посещения

### Режим "Следующее место"

- [ ] Открыть экран "Следующее место" без активной поездки (должно быть сообщение)
- [ ] Открыть экран "Следующее место" с активной поездкой
- [ ] Проверить отображение следующего непосещенного места
- [ ] Проверить расчет расстояния до места (если доступна геолокация)
- [ ] Отметить место как посещенное
- [ ] Открыть место на карте
- [ ] Открыть место в навигаторе
- [ ] Проверить случай, когда все места посещены

### Навигация

- [ ] Перейти со главного экрана во все разделы
- [ ] Вернуться назад из всех экранов
- [ ] Перейти от списка к детальному экрану
- [ ] Перейти от детального экрана к форме редактирования
- [ ] Перейти от поездки к месту в поездке

### Настройки

- [ ] Просмотреть статистику
- [ ] Просмотреть информацию о приложении
- [ ] Очистить все данные (с подтверждением)
- [ ] Проверить, что данные действительно удалены

## Тестирование на устройствах

### Android

1. Запустить приложение на Android эмуляторе или устройстве
2. Проверить работу всех функций
3. Проверить работу в офлайн режиме (отключить интернет)
4. Проверить работу с геолокацией
5. Проверить работу камеры и галереи
6. Проверить открытие в Google Maps
7. Проверить открытие в системном навигаторе

### iOS

1. Запустить приложение на iOS симуляторе или устройстве
2. Проверить работу всех функций
3. Проверить работу в офлайн режиме (отключить интернет)
4. Проверить работу с геолокацией
5. Проверить работу камеры и галереи
6. Проверить открытие в Apple Maps
7. Проверить открытие в системном навигаторе

## Тестирование офлайн режима

1. Запустить приложение с интернетом
2. Создать несколько мест и поездок
3. Добавить фотографии
4. Отключить интернет
5. Проверить:
   - Просмотр списков мест и поездок
   - Создание новых мест и поездок
   - Редактирование существующих данных
   - Просмотр фотографий
   - Работа карт (после загрузки тайлов)

## Тестирование производительности

1. Создать большое количество мест (100+)
2. Создать несколько поездок с множеством мест
3. Проверить скорость загрузки списков
4. Проверить скорость поиска и фильтрации
5. Проверить работу с большим количеством фотографий

## Тестирование валидации

### Валидация мест
- [ ] Пустое название (должна быть ошибка)
- [ ] Название слишком длинное (должна быть ошибка)
- [ ] Неверный формат координат (должна быть ошибка)
- [ ] Координаты вне допустимого диапазона (должна быть ошибка)

### Валидация поездок
- [ ] Пустое название (должна быть ошибка)
- [ ] Неверный формат даты (должна быть ошибка)
- [ ] Дата начала позже даты окончания (должна быть ошибка)

## Тестирование обработки ошибок

1. Попробовать удалить несуществующее место (должна быть ошибка)
2. Попробовать открыть место без координат в навигаторе (должна быть ошибка)
3. Попробовать добавить фотографию без разрешения (должна быть ошибка)
4. Попробовать получить геолокацию без разрешения (должна быть ошибка)

## Рекомендации по тестированию

1. Тестируйте на реальных устройствах, а не только на эмуляторах
2. Тестируйте в разных условиях (с интернетом и без)
3. Тестируйте с разными разрешениями экрана
4. Тестируйте с большим количеством данных
5. Проверяйте работу после перезапуска приложения
6. Проверяйте работу после очистки данных
